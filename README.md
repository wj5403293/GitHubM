## <img src="https://api.iconify.design/lucide:info.svg?color=%237c3aed" width="22" style="vertical-align: middle;" /> 简介

GitHubM 是一个全功能的 GitHub 管理面板，提供仓库浏览、Issue追踪、Pull Request管理、Gist、通知、搜索等一站式操作界面。支持 **Web浏览器** 和 **Android APK** 两种使用方式。

## <img src="https://api.iconify.design/lucide:rocket.svg?color=%237c3aed" width="22" style="vertical-align: middle;" /> 快速开始

### 环境要求

- **Node.js** ≥ 20
- **pnpm** ≥ 9

```bash
node -v   # v20.x+
pnpm -v   # 9.x+
```

本地开发

```bash
# 1. 克隆仓库
git clone https://github.com/qq5855144/GitHubM.git
cd GitHubM

# 2. 安装依赖
pnpm install

# 3. 启动开发服务器
pnpm exec vite
```

构建生产版本

```bash
pnpm exec vite build    # 产物输出到 dist/
```

<img src="https://api.iconify.design/lucide:git-branch.svg?color=%237c3aed" width="22" style="vertical-align: middle;" /> CI/CD 工作流

```
push main ──► build-web ──┬── deploy-pages ──► GitHub Pages
                          └── build-apk  ──► APK Artifact
```

· 触发条件：推送到 main 分支 / 手动触发
· Web 部署：Vite 构建 → actions/deploy-pages@v4 → GitHub Pages
· APK 构建：下载 Web 产物 → 复制到 Android assets → Gradle 构建 → 上传 Artifact（保留 30 天）

<img src="https://api.iconify.design/lucide:lock.svg?color=%237c3aed" width="22" style="vertical-align: middle;" /> 使用方式

1. 访问 GitHubM
2. 在 GitHub 生成 Personal Access Token（需要 repo、notifications、user 权限）
3. 输入 Token 登录，即可管理你的 GitHub 资源

<img src="https://api.iconify.design/lucide:shield-alert.svg?color=%23eab308" width="16" style="vertical-align: middle;" /> Token 仅保存在浏览器本地，不会上传至任何服务器。
